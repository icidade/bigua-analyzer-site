Why bigua-analyzer?

⚡

Fast by default

Incremental analysis so only changed files are re-checked on every run, keeping feedback loops short.

🔌

Extensible

A simple plugin API lets you write custom rules that fit your project's conventions and style guide.

🤫

Low noise

Smart suppression and severity levels so your team only sees what genuinely needs attention.

🔓

Open source

Fully open and MIT-licensed. Inspect it, fork it, contribute to it — no vendor lock-in.

🔗

CI-ready

Works out of the box with GitHub Actions, GitLab CI, and other popular pipelines via a single command.

📊

Actionable reports

Export results as JSON, HTML, or SARIF. Integrates with code review tools to surface findings inline.

Quick start (coming soon)

# install
pip install bigua-analyzer

# run against your project
bigua analyze ./src

# or integrate with GitHub Actions
bigua analyze ./src --format sarif --output results.sarif
